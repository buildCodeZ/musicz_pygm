```
键盘按键弹钢琴的小程序
安装:
    pip install musicz_pygm

程序运行:
    python -m musicz_pygm 可选参数
    参数如下:
        [-f/--fp=]额外配置文件(没啥用)
        [-t/--default=]主要配置文件(默认预制的play.js)
        [-h/--help]
        [-b/--background] 添加背景音乐
        [-w/--width] 窗口宽度
        [-h/--height] 窗口高度
        --noframe=0/1 是否隐藏窗口边框
    默认配置文件（musicz_pygm/conf目录下）：
        通用配置：
            shift和ctrl调整基调，方向键调整音量，esc和~退出，空格切音

        
```
